Git Submodules

I remember having so much trouble when I tried to build a GLFW project on my M1 mac for the first time. The easiest, most reliable cross platform way to build a GLFW project is via Git submodules tool . A tool that allows using other projects from GitHub within a project. This blog post outlines the step by step guide to building GLFW project usning CMake, git and GitHub. GLFW is used as an example but the same approach could be used to import any other library on GitHub.
